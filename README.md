# Clover Community

Это Vanilla+ сервер Minecraft, ранее известный как Erdocraft, с полуролевым форматом и акцентом на социальное взаимодействие. Здесь нет жёсткого сценария — игроки сами определяют, чем заниматься: строить города, развивать экономику, участвовать в жизни сообщества или просто играть в своём темпе.
